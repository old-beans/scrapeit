
Scrapes all text from a web address or list of web addresses, from the command line, and writes to .txt

Works with Windows, probably nothing else. 
Add the folder containing scrapeit.bat to the PATH variable, and update .bat file with the location of scrapeit.py
Add 'input' and 'output' folders to the same directory as scrapeit.py

From command line or Run: scrapeit.py
By default, scrapeit will use the url on your clipboard as the input

Optional arguments - file filename.txt
place filename.txt in input folder. Contains line-separated urls to be scraped
